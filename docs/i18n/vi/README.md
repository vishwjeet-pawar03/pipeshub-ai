<h2 id="about-pipeshub">PipesHub — Có thể giải thích & Có thể mở rộng</h2>

<strong>[PipesHub](https://www.pipeshub.com/)</strong> là một lớp thực thi gốc AI, mã nguồn mở và có thể tự lưu trữ, kết nối tri thức doanh nghiệp, cung cấp tìm kiếm có thể giải thích kèm trích dẫn và tự động hóa quy trình làm việc trên khắp các hệ thống của bạn.


## Tính năng

- 📝 **Câu trả lời có thể giải thích:** PipesHub đưa ra các câu trả lời có căn cứ kèm trích dẫn chính xác đến từng khối trong tài liệu gốc.
- 🔒 **Tìm kiếm nhận biết quyền truy cập:** Thực thi kiểm soát truy cập ở cấp nguồn để người dùng chỉ thấy những gì họ được phép.
- 🕸️ **Truy xuất bằng đồ thị tri thức:** Truy xuất dựa trên đồ thị, nắm bắt các mối quan hệ giữa dữ liệu doanh nghiệp.
- 🔌 **Trình kết nối doanh nghiệp:** Hơn 30 trình kết nối với lập chỉ mục thời gian thực và theo lịch, sẵn sàng dùng ngay.
- 🔍 **Tìm kiếm hợp nhất, Nghiên cứu sâu và Tác nhân:** Tìm kiếm, hỏi đáp, nghiên cứu sâu, tìm kiếm web và các tác nhân AI trên cùng một lớp ngữ cảnh.
- 📊 **Tạo tác phẩm và Thực thi mã:** Tạo báo cáo, biểu đồ và bảng điều khiển trong một hộp cát thực thi an toàn.
- 🎙️ **Hỗ trợ đa phương thức:** Hiểu hình ảnh, sơ đồ và tệp quét, cùng với tương tác bằng giọng nói.
- 🤖 **Tác nhân và Hành động không cần mã:** Xây dựng tác nhân một cách trực quan và thực thi hành động trên các công cụ doanh nghiệp.
- 🧠 **Mang mô hình của riêng bạn, hoàn toàn tự lưu trữ:** Bất kỳ nhà cung cấp LLM nào, triển khai trong VPC của bạn — dữ liệu không bao giờ rời khỏi hạ tầng của bạn.
- 🛠️ **Ưu tiên nhà phát triển và có thể mở rộng:** API, SDK, công cụ MCP, trình kết nối tùy chỉnh và các dịch vụ có thể mở rộng độc lập.

## 🚀 Hướng dẫn triển khai

PipesHub (Nền tảng AI cho nơi làm việc) có thể chạy cục bộ hoặc triển khai trên đám mây bằng Docker Compose.
**Lưu ý**: Nếu bạn triển khai PipesHub trên máy chủ đám mây, hãy đảm bảo sử dụng điểm cuối HTTPS. PipesHub thực thi các kiểm tra bảo mật chặt chẽ hơn, và trình duyệt sẽ chặn một số yêu cầu khi ứng dụng được phục vụ qua HTTP.
Bạn có thể dùng một reverse proxy như Cloudflare, Nginx hoặc Traefik để kết thúc SSL/TLS và cung cấp chứng chỉ HTTPS hợp lệ.
Nếu bạn thấy màn hình trắng sau khi triển khai PipesHub khi truy cập qua HTTP, đây rất có thể là nguyên nhân. Frontend sẽ từ chối tải do các kiểm tra bảo mật chặt chẽ hơn.

---

### 📦 Triển khai sản xuất

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git

# 📁 Navigate to the deployment folder
cd pipeshub-ai/deployment/docker-compose

# Set Environment Variables
> 👉 Set Environment Variables for secrets, passwords, and the public URLs of the **Frontend** and **Connector** services
> _(Required for webhook notifications and real-time updates)_
> Refer to env.template

# 🚀 Start the production deployment
docker compose -f docker-compose.prod.yml -p pipeshub-ai up -d

# 🛑 To stop the services
docker compose -f docker-compose.prod.yml -p pipeshub-ai down
```

### 📦 Bản dựng triển khai cho nhà phát triển

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git

# 📁 Navigate to the deployment folder
cd pipeshub-ai/deployment/docker-compose

# Set Optional Environment Variables
> 👉 Set Environment Variables for secrets, passwords, and the public URLs of the **Frontend** and **Connector** services
> _(Required for webhook notifications and real-time updates)_
> Refer to env.template

# 🚀 Start the local build deployment
docker compose -f docker-compose.build.neo4j.yml -p pipeshub-ai up --build -d

# 🛑 To stop the services
docker compose -f docker-compose.build.neo4j.yml -p pipeshub-ai down
```

`Dockerfile` chính kéo các lớp dựng sẵn từ `pipeshubai/pipeshub-ai-base:python-deps` và `pipeshubai/pipeshub-ai-base:runtime` (xem [`Dockerfile.base`](../../../Dockerfile.base) ở thư mục gốc của kho lưu trữ để biết lệnh build/push). Để dùng thẻ cục bộ thay thế, hãy đặt `PYTHON_DEPS_IMAGE` và `RUNTIME_BASE_IMAGE` trong môi trường hoặc trong tham số build của compose.

## Câu hỏi thường gặp

### PipesHub là gì?

PipesHub là một lớp thực thi gốc AI, mã nguồn mở và có thể tự lưu trữ, kết nối tri thức doanh nghiệp, cung cấp tìm kiếm có thể giải thích kèm trích dẫn và tự động hóa quy trình làm việc trên khắp các hệ thống của bạn. Nó cung cấp một lớp ngữ cảnh hợp nhất cho tìm kiếm, hỏi đáp, nghiên cứu sâu, tìm kiếm web và các tác nhân AI.

### PipesHub khác với các công cụ AI cho nơi làm việc khác như thế nào?

PipesHub hoàn toàn mã nguồn mở (Apache 2.0) và có thể tự lưu trữ — dữ liệu của bạn không bao giờ rời khỏi hạ tầng của bạn. Nó có tính năng tìm kiếm nhận biết quyền truy cập, thực thi kiểm soát truy cập ở cấp nguồn, và đưa ra các câu trả lời có thể giải thích kèm trích dẫn chính xác đến từng khối trong tài liệu gốc.

### PipesHub hỗ trợ những trình kết nối nào?

PipesHub có hơn 30 trình kết nối doanh nghiệp với lập chỉ mục thời gian thực và theo lịch. Nó hỗ trợ các định dạng tệp như PDF, Docx, XLSX, PPTX, CSV, Markdown, HTML, Google Docs/Sheets/Slides, hình ảnh, âm thanh và video.

### Tôi triển khai PipesHub như thế nào?

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git
cd pipeshub-ai/deployment/docker-compose

# Set Environment Variables (refer to env.template)
# Start production deployment
docker compose -f docker-compose.prod.yml -p pipeshub-ai up -d
```

Lưu ý: Hãy dùng HTTPS cho việc triển khai trên đám mây. HTTP có thể gây ra việc chặn bảo mật ở frontend.

### PipesHub hỗ trợ những nhà cung cấp LLM nào?

PipesHub theo nguyên tắc "Mang mô hình của riêng bạn" — bạn có thể dùng bất kỳ nhà cung cấp LLM nào. Triển khai trong VPC của bạn với các mô hình bạn ưa thích. Ngăn xếp công nghệ bao gồm LangChain và LangGraph cho các pipeline và quy trình LLM.

### Tính năng truy xuất bằng đồ thị tri thức là gì?

PipesHub sử dụng truy xuất dựa trên đồ thị để nắm bắt các mối quan hệ giữa dữ liệu doanh nghiệp. Nó dùng Neo4j hoặc ArangoDB làm cơ sở dữ liệu đồ thị, kết hợp với Qdrant để tìm kiếm theo độ tương đồng vectơ.

### PipesHub có máy chủ MCP không?

Có. PipesHub cung cấp một máy chủ MCP để tích hợp với bất kỳ client tương thích MCP nào. Kho lưu trữ: [pipeshub-ai/mcp-server](https://github.com/pipeshub-ai/mcp-server/).

### Có những SDK nào?

PipesHub cung cấp SDK cho:
- **Python**: [pipeshub-ai/pipeshub-sdk-python](https://github.com/pipeshub-ai/pipeshub-sdk-python)
- **TypeScript**: [pipeshub-ai/pipeshub-sdk-typescript](https://github.com/pipeshub-ai/pipeshub-sdk-typescript)
- **Go**: [pipeshub-ai/pipeshub-sdk-go](https://github.com/pipeshub-ai/pipeshub-sdk-go)

### Tôi có thể xây dựng tác nhân AI mà không cần lập trình không?

Có. PipesHub có một trình tạo tác nhân không cần mã. Bạn có thể xây dựng tác nhân một cách trực quan và thực thi hành động trên các công cụ doanh nghiệp mà không cần viết mã.

### Hỗ trợ đa phương thức là gì?

PipesHub hỗ trợ hiểu hình ảnh, sơ đồ và tệp quét, cùng với tương tác bằng giọng nói. Nó dùng Docling và PyMuPDF để phân tích tài liệu, và Azure Document Intelligence hoặc một LLM đa phương thức (VLM) để OCR các PDF đã quét.

### Tôi khắc phục sự cố triển khai như thế nào?

1. Đảm bảo HTTPS được cấu hình cho việc triển khai trên đám mây
2. Kiểm tra nhật ký Docker compose: `docker compose logs`
3. Xác minh các biến môi trường trong env.template
4. Tham khảo [docs.pipeshub.com](https://docs.pipeshub.com/) để có hướng dẫn chi tiết
