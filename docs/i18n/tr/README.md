<h2 id="about-pipeshub">PipesHub — Açıklanabilir ve Genişletilebilir</h2>

<strong>[PipesHub](https://www.pipeshub.com/)</strong>, kurumsal bilgiyi birbirine bağlayan, kaynak göstererek açıklanabilir arama sunan ve sistemleriniz genelinde iş akışlarını otomatikleştiren, açık kaynaklı ve kendi sunucunuzda barındırabileceğiniz yapay zekâ-yerel bir yürütme katmanıdır.


## Özellikler

- 📝 **Açıklanabilir Yanıtlar:** PipesHub, orijinal belgelere kesin blok alıntıları içeren, dayanağı olan yanıtlar sunar.
- 🔒 **İzin Farkında Arama:** Kaynak düzeyinde erişim denetimlerini uygular; böylece kullanıcılar yalnızca yetkili oldukları içeriği görür.
- 🕸️ **Bilgi Grafiği ile Erişim:** Kurumsal veriler arasındaki ilişkileri yakalayan, grafik tabanlı erişim.
- 🔌 **Kurumsal Bağlayıcılar:** Gerçek zamanlı ve zamanlanmış indeksleme ile kutudan çıkar çıkmaz kullanılabilen 30'dan fazla bağlayıcı.
- 🔍 **Birleşik Arama, Derin Araştırma ve Ajanlar:** Tek bir bağlam katmanı üzerinde arama, soru-cevap, derin araştırma, web araması ve yapay zekâ ajanları.
- 📊 **Yapıtlar ve Kod Yürütme:** Güvenli bir yürütme korumalı alanında raporlar, grafikler ve panolar oluşturun.
- 🎙️ **Çok Modlu Destek:** Görüntü, diyagram ve taranmış dosya anlama ile birlikte sesli etkileşim.
- 🤖 **Kodsuz Ajanlar ve Eylemler:** Ajanları görsel olarak oluşturun ve kurumsal araçlar genelinde eylemleri yürütün.
- 🧠 **Kendi Modelini Getir, Tamamen Kendi Sunucunda Barındırılabilir:** Herhangi bir LLM sağlayıcısı, kendi VPC'nizde dağıtılır — veriler asla altyapınızdan ayrılmaz.
- 🛠️ **Geliştirici Odaklı ve Genişletilebilir:** API'ler, SDK'lar, MCP araçları, özel bağlayıcılar ve bağımsız olarak ölçeklenebilen hizmetler.

## 🚀 Dağıtım Kılavuzu

PipesHub (İş Yeri Yapay Zekâ Platformu) yerel olarak çalıştırılabilir veya Docker Compose kullanılarak bulutta dağıtılabilir.
**Not**: PipesHub'ı bir bulut sunucusunda dağıtıyorsanız, bir HTTPS uç noktası kullandığınızdan emin olun. PipesHub daha sıkı güvenlik denetimleri uygular ve uygulama HTTP üzerinden sunulduğunda tarayıcılar bazı istekleri engeller.
SSL/TLS'i sonlandırmak ve geçerli bir HTTPS sertifikası sağlamak için Cloudflare, Nginx veya Traefik gibi bir ters proxy kullanabilirsiniz.
PipesHub'ı dağıttıktan sonra HTTP üzerinden erişirken beyaz bir ekran görüyorsanız, bunun nedeni büyük olasılıkla budur. Daha sıkı güvenlik denetimleri nedeniyle ön uç yüklenmeyi reddeder.

---

### 📦 Üretim Dağıtımı

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git

# 📁 Navigate to the deployment folder
cd pipeshub-ai/deployment/docker-compose

# Set Environment Variables
> 👉 Set Environment Variables for secrets, passwords, and the public URLs of the **Frontend** and **Connector** services
> _(Required for webhook notifications and real-time updates)_
> Refer to env.template

# 🚀 Start the production deployment
docker compose -f docker-compose.prod.yml -p pipeshub-ai up -d

# 🛑 To stop the services
docker compose -f docker-compose.prod.yml -p pipeshub-ai down
```

### 📦 Geliştirici Dağıtım Derlemesi

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git

# 📁 Navigate to the deployment folder
cd pipeshub-ai/deployment/docker-compose

# Set Optional Environment Variables
> 👉 Set Environment Variables for secrets, passwords, and the public URLs of the **Frontend** and **Connector** services
> _(Required for webhook notifications and real-time updates)_
> Refer to env.template

# 🚀 Start the local build deployment
docker compose -f docker-compose.build.neo4j.yml -p pipeshub-ai up --build -d

# 🛑 To stop the services
docker compose -f docker-compose.build.neo4j.yml -p pipeshub-ai down
```

Ana `Dockerfile`, `pipeshubai/pipeshub-ai-base:python-deps` ve `pipeshubai/pipeshub-ai-base:runtime` adreslerinden önceden derlenmiş katmanları çeker (derleme/gönderme komutları için depo kökündeki [`Dockerfile.base`](../../../Dockerfile.base) dosyasına bakın). Bunun yerine yerel etiketler kullanmak için ortamda veya compose derleme argümanlarında `PYTHON_DEPS_IMAGE` ve `RUNTIME_BASE_IMAGE` değişkenlerini ayarlayın.

## SSS

### PipesHub nedir?

PipesHub, kurumsal bilgiyi birbirine bağlayan, kaynak göstererek açıklanabilir arama sunan ve sistemleriniz genelinde iş akışlarını otomatikleştiren, açık kaynaklı ve kendi sunucunuzda barındırabileceğiniz yapay zekâ-yerel bir yürütme katmanıdır. Arama, soru-cevap, derin araştırma, web araması ve yapay zekâ ajanları için birleşik bir bağlam katmanı sağlar.

### PipesHub diğer iş yeri yapay zekâ araçlarından nasıl ayrılır?

PipesHub tamamen açık kaynaklıdır (Apache 2.0) ve kendi sunucunuzda barındırılabilir — verileriniz asla altyapınızdan ayrılmaz. Kaynak düzeyinde erişim denetimlerini uygulayan izin farkında bir arama sunar ve orijinal belgelere kesin blok alıntıları içeren açıklanabilir yanıtlar verir.

### PipesHub hangi bağlayıcıları destekler?

PipesHub'ın gerçek zamanlı ve zamanlanmış indekslemeli 30'dan fazla kurumsal bağlayıcısı vardır. PDF, Docx, XLSX, PPTX, CSV, Markdown, HTML, Google Docs/Sheets/Slides, görüntü, ses ve video gibi dosya biçimlerini destekler.

### PipesHub'ı nasıl dağıtırım?

```bash
# Clone the repository
git clone https://github.com/pipeshub-ai/pipeshub-ai.git
cd pipeshub-ai/deployment/docker-compose

# Set Environment Variables (refer to env.template)
# Start production deployment
docker compose -f docker-compose.prod.yml -p pipeshub-ai up -d
```

Not: Bulut dağıtımları için HTTPS kullanın. HTTP, ön uçta güvenlik engellemelerine yol açabilir.

### PipesHub hangi LLM sağlayıcılarını destekler?

PipesHub "Kendi Modelini Getir" yaklaşımını benimser — herhangi bir LLM sağlayıcısını kullanabilirsiniz. Tercih ettiğiniz modellerle kendi VPC'nizde dağıtım yapın. Teknoloji yığını, LLM ardışık düzenleri ve iş akışları için LangChain ve LangGraph içerir.

### Bilgi Grafiği ile Erişim özelliği nedir?

PipesHub, kurumsal veriler arasındaki ilişkileri yakalayan grafik tabanlı erişimi kullanır. Grafik veritabanı olarak Neo4j veya ArangoDB'yi, vektör benzerliği araması için Qdrant ile birlikte kullanır.

### PipesHub'ın bir MCP sunucusu var mı?

Evet. PipesHub, MCP uyumlu herhangi bir istemciyle entegrasyon için bir MCP sunucusu sağlar. Depo: [pipeshub-ai/mcp-server](https://github.com/pipeshub-ai/mcp-server/).

### Hangi SDK'lar mevcut?

PipesHub şunlar için SDK sağlar:
- **Python**: [pipeshub-ai/pipeshub-sdk-python](https://github.com/pipeshub-ai/pipeshub-sdk-python)
- **TypeScript**: [pipeshub-ai/pipeshub-sdk-typescript](https://github.com/pipeshub-ai/pipeshub-sdk-typescript)
- **Go**: [pipeshub-ai/pipeshub-sdk-go](https://github.com/pipeshub-ai/pipeshub-sdk-go)

### Kod yazmadan yapay zekâ ajanları oluşturabilir miyim?

Evet. PipesHub'ın kodsuz bir ajan oluşturucusu vardır. Ajanları görsel olarak oluşturabilir ve kod yazmadan kurumsal araçlar genelinde eylemleri yürütebilirsiniz.

### Çok modlu destek nedir?

PipesHub, görüntü, diyagram ve taranmış dosya anlamanın yanı sıra sesli etkileşimi destekler. Belge ayrıştırma için Docling ve PyMuPDF'i, taranmış PDF OCR'si için Azure Document Intelligence veya çok modlu bir LLM (VLM) kullanır.

### Dağıtım sorunlarını nasıl giderebilirim?

1. Bulut dağıtımları için HTTPS'in yapılandırıldığından emin olun
2. Docker compose günlüklerini kontrol edin: `docker compose logs`
3. env.template içindeki ortam değişkenlerini doğrulayın
4. Ayrıntılı kılavuzlar için [docs.pipeshub.com](https://docs.pipeshub.com/) adresine başvurun
