## 私たちの誓い

私たちはメンバー、貢献者、リーダーとして、年齢、体格、目に見える障害・見えない障害、民族性、性的特徴、性自認と性表現、経験の水準、教育、社会経済的地位、国籍、容姿、人種、宗教、性的指向とアイデンティティにかかわらず、誰にとってもハラスメントのない体験となるコミュニティへの参加を実現することを誓います。

私たちは、オープンで、歓迎的で、多様で、包摂的かつ健全なコミュニティに資する形で行動し、交流することを誓います。

## 私たちの基準

コミュニティにとって前向きな環境に寄与する行動の例には、次のものが含まれます：

* 他者への共感と思いやりを示すこと
* 異なる意見、視点、経験を尊重すること
* 建設的なフィードバックを与え、また快く受け入れること
* 自らの過ちに対して責任を受け入れ、影響を受けた人々に謝罪し、その経験から学ぶこと
* 個人としての自分だけでなく、コミュニティ全体にとって最善であることに焦点を当てること

容認できない行動の例には、次のものが含まれます：

* 性的な言葉や画像の使用、およびあらゆる種類の性的な関心の表明や言い寄り
* 荒らし行為、侮辱的または中傷的なコメント、ならびに個人攻撃や政治的攻撃
* 公的または私的なハラスメント
* 明示的な許可なく、住所やメールアドレスなど他者の私的情報を公開すること
* 専門的な場において不適切と合理的に判断されうるその他の行為

## 執行の責任

コミュニティリーダーは、容認される行動の基準を明確にし、それを執行する責任を負い、不適切、脅迫的、攻撃的、または有害とみなされるあらゆる行動に対して、適切かつ公正な是正措置を講じます。

コミュニティリーダーは、本行動規範に沿わないコメント、コミット、コード、wiki の編集、issue、その他の貢献を削除、編集、または却下する権利と責任を有し、適切な場合にはモデレーションの判断の理由を伝えます。

## 適用範囲

本行動規範は、すべてのコミュニティ空間に適用され、また個人がコミュニティを公的な場で公式に代表する場合にも適用されます。コミュニティを代表する例には、公式のメールアドレスの使用、公式のソーシャルメディアアカウントを通じた投稿、オンラインまたはオフラインのイベントでの任命された代表としての活動が含まれます。

## 執行

虐待的、嫌がらせ的、またはその他の容認できない行動については、執行を担当するコミュニティリーダーへ contact@pipeshub.com にて報告できます。
すべての苦情は速やかかつ公正に審査・調査されます。

すべてのコミュニティリーダーは、いかなる事案においても、報告者のプライバシーと安全を尊重する義務を負います。

## 執行ガイドライン

コミュニティリーダーは、本行動規範に違反するとみなした行為の結果を判断する際、次の「コミュニティへの影響に関するガイドライン」に従います：

### 1. 是正

**コミュニティへの影響**：不適切な言葉遣い、またはコミュニティにおいて専門的でない、もしくは歓迎されないとみなされるその他の行為。

**結果**：コミュニティリーダーからの私的かつ書面による警告。違反の性質を明確にし、その行為がなぜ不適切であったかを説明します。公的な謝罪が求められる場合があります。

### 2. 警告

**コミュニティへの影響**：単一の事案または一連の行為による違反。

**結果**：継続した場合の結果を伴う警告。一定期間、関係者との交流（本行動規範を執行する人々への一方的な働きかけを含む）を行わないこと。これには、コミュニティ空間だけでなく、ソーシャルメディアなどの外部チャネルでの交流を避けることも含まれます。これらの条件に違反した場合、一時的または恒久的な追放につながる可能性があります。

### 3. 一時的追放

**コミュニティへの影響**：持続的な不適切行為を含む、コミュニティ基準の重大な違反。

**結果**：一定期間、コミュニティとのあらゆる種類の交流または公的なコミュニケーションを禁止します。この期間中、関係者との公的または私的な交流（本行動規範を執行する人々への一方的な働きかけを含む）は一切許可されません。これらの条件に違反した場合、恒久的な追放につながる可能性があります。

### 4. 恒久的追放

**コミュニティへの影響**：持続的な不適切行為、個人へのハラスメント、または特定の集団に対する攻撃や蔑視を含む、コミュニティ基準の違反パターンを示すこと。

**結果**：コミュニティ内でのあらゆる種類の公的な交流の恒久的な禁止。
