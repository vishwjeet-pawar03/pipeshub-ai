💻 Day 2 — Dev Environment

Prerequisites

Confirm these are installed before cloning:

  • Node.js 20+ — node --version
  • Python 3.12+ — python3 --version
  • Docker Desktop (must be running)
  • git configured with your @pipeshub.com email

Setup steps

  1. Clone the repository
    bash
    git clone git@github.com:pipeshub/pipeshub-ai.git
cd pipeshub-ai
cp .env.example .env.local

    Fill in .env.local from the values pinned in #eng-onboarding. Don't commit it.

  2. Install dependencies
    bash
    # Node / TypeScript
npm install

# Python service layer
pip install -r services/requirements.txt --break-system-packages
  3. Start all services
    bash
    docker compose up -d
npm run dev

# In a second terminal:
cd services && uvicorn main:app --reload

    App at http://localhost:3000 · API at http://localhost:8000

  4. Run the test suites — all should pass on a clean checkout
    bash
    npm test                   # TypeScript unit tests
cd services && pytest       # Python tests
npm run test:integration    # Requires Docker

Pull request rules

  • Branch naming: your-name/short-description (e.g., arjun/fix-s3-content-type)
  • PR must have a description, Linear issue link, and at least one test change
  • Request review from ≥1 person; ≥2 for anything touching auth or credentials
  • Squash-merge to main
  • CI must be green before merging — no exceptions
